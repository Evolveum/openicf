Exchange Connector Overview

The Exchange Connector supports provisioning to Windows servers running Active Directory and MS Exchange Server 2007. In fact, the Exchange Connector is and extension of the Active Directory connector.

Bundle Information

  • Latest stable release: 1.0.4559
  • Platform: .NET
  • 3rd-party libraries required: None
  • Other dependencies: MS PowerShell 1.0/MS Exchange 2007

Configuration Properties

The properties are same as for the Active Directory connector.

Schema and Supported Features

The following lists all explicitly supported attributes in the schema by Exchange Connector, it expands the attributes supported by Active Directory connector. Many attributes beyond these can be set as well. Unless othewise noted, all attributes are Createable, Updateable, Readable, and Single Value

For ObjectClass ACCOUNT, the following additional attributes are supported:

  • RecipientType (String) : REQUIRED (possible values: User, MailUser, UserMailbox)
  • ExternalEmailAddress (String) : MULTIVALUE
  • Database (String) :

    The Exchange Connector supports the same operations as Active Directory connector.
