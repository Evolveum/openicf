Configuration Properties

These are the properties that can be specified on the Configuration object when setting up the Connector.

  • Required:
    • hostNameOrIpAddr (String): Host address of the VMS system being managed
    • userName (String): User name of the management account
    • password (GuardedString): Password of the management account
    • hostPortNumber (String): Port number used to connect to the VMS system being managed
    • isSSH (Boolean): Whether to use SSH to connect to the VMS system being managed
    • hostLineTerminator (String): A string indicating the line terminator:
      • \r\n is the normal line terminator for a Telnet connection
      • \r is the normal line terminator for an SSH session
    • hostShellPrompt (String): An Expect4J regular expression to match the VMS prompt:
      • default is "[$] ", which is the VMS standard
    • vmsTimeZone (String): Time zone in which the VMS system is located (expressed as a valid Java TimeZone name)
    • disableUserLogins (Boolean): Whether to disable the connector from logging in on the VMS system being managed as the user being managed. Setting this to false disables:
      • running scripts as user
      • changing password as user
      • authorize
      However, if user logins are allowed, VMS breakin evasion may disable the connector, depending on the security settings on the VMS system being managed
  • Optional (these are used in not-yet-tested code to allow support for non-US locales):
    • vmsLocale (String): (defaults to en_US) The locale used on the VMS system.
    • vmsDateFormatWithSecs (String): (defaults to dd-MMM-yyyy HH:mm:ss) The format used for time-stamps with seconds on the VMS system.
    • vmsDateFormatWithoutSecs (String): (defaults to dd-MMM-yyyy HH:mm) The format used for time-stamps without seconds on the VMS system.

Schema and Supported Features

This Connector supports the ACCOUNT object class. The attributes supported are:

  • ACCESS
  • ACCOUNT
  • ASTLM
  • BATCH
  • BIOLM
  • BYTLM
  • CLI
  • CLITABLES
  • CPUTIME
  • DEFPRIVILEGES
  • DEFAULT
  • DEVICE
  • DIALUP
  • DIOLM
  • DIRECTORY
  • ENQLM
  • EXPIRATION
  • FILLM
  • FLAGS
  • GENERATE_PASSWORD
  • INTERACTIVE
  • JTQUOTA
  • LGICMD
  • LOCAL
  • LoginFails
  • MAXACCTJOBS
  • MAXDETACH
  • MAXJOBS
  • MODIFY_IDENTIFIER
  • NETWORK
  • OWNER
  • PBYTLM
  • PGFLQUOTA
  • PRCLM
  • PRIMEDAYS
  • PRIORITY
  • PRIVILEGES
  • PWDEXPIRED
  • PWDMINIMUM
  • QUEPRIO
  • REMOTE
  • SHRFILLM
  • TQELM
  • UIC
  • WSDEFAULT
  • WSEXTENT
  • WSQUOTA
  • COPY_LOGIN_SCRIPT
  • CREATE_DIRECTORY
  • LOGIN_SCRIPT_SOURCE
Furthermore, the ACCOUNT object class supports the following special attributes:
  • OperationalAttributeInfos.PASSWORD
  • OperationalAttributeInfos.PASSWORD_EXPIRED
  • OperationalAttributeInfos.CURRENT_PASSWORD
  • OperationalAttributeInfos.ENABLE
  • PredefinedAttributeInfos.LAST_PASSWORD_CHANGE_DATE
  • PredefinedAttributeInfos.LAST_LOGIN_DATE
  • PredefinedAttributeInfos.PASSWORD_CHANGE_INTERVAL
The VMS Connector supports the following operations:
  • Authenticate
  • Create
  • Delete
  • Update
  • Search
  • Schema
  • ScriptOnResource
