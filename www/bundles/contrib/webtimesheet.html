WebTimeSheet Connector Overview

The WebTimesheet Connector is can be used for provisioning of Replicon Web TimeSheet

Configuration Properties

These are the properties that can be specified on the Configuration object when setting up the Connector.

  • Optional:
    • URLProperty (String): URL of Web TimeSheet instance (ie: http://hosted.webtimesheet.com/mycompany/replicon or http://myserver/cgi/rt.dll)
    • AdminUidProperty(String): Administrator user ID
    • AdminPasswordProperty(String): WTS Administrator user password
    • AppPasswordProperty(String): WTS Integration Application password - the connector will register with this password on first connection

Schema and Supported Features

This Connector supports the ACCOUNT object class and "Department" object class. The ACCOUNT object class supports the following special attributes:

  • FirstName
  • LastName
  • LoginName
  • Id
  • EmployeeId
  • InternalEmail
  • ParentDepartmentId
  • DepartmentId
  • Domain
  • AuthenticationType
The WebTimeSheet Connector supports the following operations:
  • Create
  • Delete
  • Update
  • Search
  • Schema
  • Test
