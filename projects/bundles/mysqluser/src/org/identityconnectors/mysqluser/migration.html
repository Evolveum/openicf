Differences between MySQLResourceAdapter and MySQLUserConnector

Run the conversion task

The new "MySQL Conversion Utility" server task will convert adapter to connector.

Configuration properties

All configuration properties are supported except the "JDBC URL Template".
The "JDBC URL Template" is the same for all supported MySQL databases. Lets set to the MySQLUserConector configuration the same values as they are in the Adapter.
The newly supported are DataSource name and jndiProperties.
When new datasource is configured, the user, host, password and port could be left empty.
